# UNAL Student Retention Analytics (Public Data)

**Demo (GitHub Pages):** (pendiente)  
**Repo:** https://github.com/rarozomun/unal-retention-analytics

## Objetivo
Construir un case study de analítica (ETL → KPIs → visualización) con **datos públicos** sobre **retención/permanencia estudiantil** en la Universidad Nacional de Colombia (UNAL).

## Pregunta guía (MVP)
¿En qué sedes/programas/periodos se observan patrones de mayor NO RETENCIÓN y qué señales observables ayudan a priorizar acciones de permanencia?
